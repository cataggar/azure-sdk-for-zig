# azure_sdk_core_tracing

Tracing abstractions for the Azure SDK for Zig, including `Span`, `Tracer`, and
`NoopTracer`.

- Source: `sdk/core/tracing`
- Release branch: `sdk/core_tracing`
- Initial version: `0.1.0`
- Internal dependencies: none

This module is currently built by the root workspace and will become an
independent package before `azure_sdk_core`.
